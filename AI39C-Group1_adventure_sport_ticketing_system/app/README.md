# Sport Adventure Web Application

This directory contains the source code for the Sport Adventure web application, organized using the Model-View-Controller (MVC) pattern and Flask.

## Structure
- `run.py`: The entry point for starting the development server.
- `app/`: Main application directory.
  - `controllers/`: Handles incoming requests and coordinates logic.
  - `models/`: Database connection and data structures.
  - `routes/`: Routing definition for pages and API endpoints.
  - `static/`: Frontend static files (CSS, JS, assets).
  - `templates/`: Jinja2 HTML templates.
